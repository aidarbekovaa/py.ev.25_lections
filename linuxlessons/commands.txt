Linux commands

ctrl+alt+t - команда открытия терминала

~ - символ домашней директории

/ - символ корневой директории

.. - обозначает предыдущие директории
 
. - обозначает текущую директорию

 1. ls(list) - команда для вывода всех директорий(папка) и файлов в текущей(рабочей) директории в терминале  
 
 2. cd(change directory) - команда для извенения(переходов) одной директории на другую
       cd <название директории>
       
 3. mkdir(make directory) - создает указанную директорию в рабочей директории в терминале
 
 4. touch - создает указанный файл 
 
 5. rm(remove) - удаление файлов
 
 6. rmdir(remove directory) - удаление директории
 
 7. pwd(present working directory) - выводит путь до текущей директории
 
  8. rm -rf - команда принудительного удаления
  
  9. ls -a - выводит список скрытых так и обычных файлов и директорий 
  
  10. nano <path_to_file> - откроется файл для редактирования в графическом интерфейсе *если нет файла, ещ nano попытается создать новый файл
  
  11. cat<file_name> - выводит содержимое файла в терминал
